# **App Name**: Criador de Aplicativos

## Core Features:

- Definir nome do aplicativo: Permitir que os usuários definam o nome do aplicativo.
- Escrever descrição do aplicativo: Permitir que os usuários criem a descrição do aplicativo.
- Recursos principais de sugestões: Fornecer recursos principais sugeridos, alimentados por IA generativa com base no nome e na descrição do aplicativo, para ajudar a acelerar o processo de definição de recursos. A IA generativa será usada como uma ferramenta de auxílio e para ajudar o usuário. Isso aumentará a eficiência.
- Salvar detalhes do aplicativo: Permitir que os usuários salvem todos os detalhes do aplicativo.
- Interface de usuário de exibição: Exiba a interface de usuário do aplicativo.

## Style Guidelines:

- Cor primária: roxo suave (#A088F9) para criatividade e elegância.
- Cor de fundo: roxo claro dessaturado (#F4F2FC) para uma tela suave.
- Cor de destaque: Rosa vivo (#F988D8) para ações e interação.
- Fonte: 'Poppins' (sans-serif) para um visual moderno e fácil de ler. Perfeito para títulos e textos curtos.
- Use ícones modernos e criativos relacionados a aplicativos e desenvolvimento.
- Layout limpo e moderno com espaço em branco generoso para melhor usabilidade.
- Animações suaves e sutis para gerar recursos de destaque e facilitar as interações do usuário.
